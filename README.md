# billiww3
Этот бот представляет собой многофункциональный Telegram-помощник для покупки одежды. Он разработан для пользователей которые хотят выделяться за счет внешнего вида.
Основной функционал: Регистрация пользователей: Приветственное сообщение и обязательная регистрация по команде /register. Хранение данных пользователей (имя, ID, статус бана) в базе данных SQLite.
Главное меню (интерактивные кнопки): После регистрации пользователю доступны 5 основных функций:
1️ Корзина: Пользователь может видеть свои оплаченные/ не оплаченные покупки
2️ Каталог: Выбор одежды на свой вкус, от шляп до обуви
3️ Бонусы: есть шанс на скидку от 1-20%
4️ Профиль: показывает ваш зарегистрированный профиль
5 Админ-панель (команда /admin): Доступна только для заранее указанных ADMIN_IDS. Позволяет просматривать список последних зарегистрированных пользователей (с указанием статуса бана). Дает возможность забанить или разбанить пользователя по его Telegram ID.
6 Система безопасности: Проверка пользователя при каждом действии: забанен ли он и зарегистрирован ли. Заблокированные пользователи не могут взаимодействовать с ботом. Обработка ошибок:
Реагирует на неизвестные команды и сообщения, возвращая пользователя в главное меню.
Команда /cancel позволяет выйти из любого диалога (ConversationHandler).
Техническая реализация: Бот написан на Python с использованием библиотеки python-telegram-bot (v20+), использует SQLite для хранения данных и ConversationHandler для организации многошаговых диалогов.
